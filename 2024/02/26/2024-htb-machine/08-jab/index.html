Shell in the box

既然可以執行一個shell，那就可以放一個reverse tcp shell。

枚舉了一下，看到了端口 9090 有一個服務開著，然後本地轉發看看：

打開：http://127.0.0.1:9090/​， 使用 svc_openfire​ 的賬號登陸進去，看到版本：Openfire 4.7.5, build ee4395e​，

於是上網搜索 Openfire 4.7.5 exploit​ ，找到 CVE-2023-32315​，然後搜索： CVE-2023-32315 github POC​ ，找到這個：

  1. Run exploit
  2. login with newly added user
  3. goto tab plugin > upload plugin openfire-management-tool-plugin.jar​
  4. goto tab server > server settings > Management tool

https://github.com/miko550/CVE-2023-32315

根據提示只需要把插件上傳上去就可以了。
