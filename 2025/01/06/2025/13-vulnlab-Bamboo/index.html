VulnLab - Machine - Bamboo

manesec
2025-01-06

​​

https://api.vulnlab.com/api/v1/share?id=d018f972-fc9a-496b-8734-630b73c904b9

首先，修改單線程的 spose.py 工具，使其支持多線程，從而加快了端口掃描的速度，迅速識別出 9100-9200 範圍內的開放端口。接著，通過配置代理和使用 FoxyProxy 插件，作者成功訪問了 PaperCut NG 22.0 版本的服務。隨後，作者發現了兩個可用的漏洞：CVE-2023-27350 和一個繞過登錄的漏洞。利用 CVE-2023-27350，作者上傳了一個 payload，成功獲得了遠程代碼執行（RCE）的能力。通過另一個漏洞，作者能夠繞過登錄，直接進入後台管理界面。在後台，作者使用 pspy 工具監控進程，發現某些功能以 root 權限運行。最終，通過在後台點擊特定功能，作者成功地以 root 權限運行 shell。

Nmap scan

Enumerating ports via squid

使用100個綫程幾分鐘就出結果了：

所以修改一下 proxychains​ 的配置文件，在最底部加上 http 10.10.74.39 3128​，然後掃一下9100-9200​的端口：

Connect with http proxy and found PaperCut NG
